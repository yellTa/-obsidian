# SRP
한 클래스는 하나의 책임만 가져야한다.
- 하나의 책임은 모호하다
	- 클 수 있고, 작을 수 있다.
	- 문맥과 상황에 따라서 다르다.
- 중요한 기준은 변경 -> 변경이 있있을 때 파급 효과가 적으면 단일 책임 원칙을 잘 따른 것
