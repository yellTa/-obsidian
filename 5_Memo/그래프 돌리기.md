# 그래프 돌리기
그래프 문제를 보면 열과 행이 바뀌는 경우가 종종 존재한다. 
바로 
[일루미네이션](https://www.acmicpc.net/problem/5547)요런 문제류? 받을 때 행과 열이 뒤바뀌는 경우가 있다. 

![[Pasted image 20240809022858.png]]
이런 느낌? 하지만 사실 직접 구현해보면서 부딪히는게 제일 낫지 않나 싶은 생각이 든다.
