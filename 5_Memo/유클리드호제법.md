# 나의 생각
## 유클리드 호제법
두 수의 최대 공약수를 구하는 알고리즘이다.
[[MOD연산]]연산을 필요로 한다. 

```
int gcd(int a, int b) {
    int c = 0;
    while(true) {
        c = a % b;
        if(c == 0) {
            break;
        }
        a = b;
        b = c;
    }
    return b;
}
```


## 최소 공배수를 구하기
> 두 수의 최대 공약수 * 두 수의 최소 공배수= 두 수의 곱

## 세 수의 공배수를 구하기
두 수를 묶어서 계산한다.
큰 수 -> 작은 수로 묶으면 되고 나머지 연산을 계쏙 수행하면 된다.
