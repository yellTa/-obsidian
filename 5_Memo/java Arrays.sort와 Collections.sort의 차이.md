# Java Arrays.sort()와 Collections.sort의 차이

# 설명
## Arrays.sort()
평균 시간 복잡도 ->O(NlogN)
최악은 O(N^2)

따라서 시간초과가 날 가능성이 있다.

## Collections.sort()
O(N)~ O(NlogN)의 시간 복잡도
Collections.sort는 primitive타입(기본타입)이 아닌 List계열의 자료구조를 사용해야한다.
