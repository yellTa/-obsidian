# ArrayList와 LinkedList의 차이

## 설명
### LInkedList
- LinkedList - > O(n) ArrayList보다 작다.
- 요소 삽입시 객체를 생성하고 서로 참조한다. 이 과정에서 메모리를 사용한다.
### ArrayList
- ArrayList는 기본적으로 배열 구조이다.


n이 10만 100만 억으로 올라가게 되면 LinekdList가 유리하게 된다. (연산 수행이 많이 일어날 때 O(n)이라서 )
