# 비트연산자란?

# 설명
이진수로 표현된 데이터에 대한 연산을 수행한다.

## AND &
0011 & 0001 = 0001

## OR |
0001 | 0011 = 0011

## XOR ^
다를 때 1이 된다.
0101 ^ 0011 = 0110

## NOT ~
비트 반전
0101~ = 1010

## <<
왼쪽으로 부호비트를 이동시킨다.
```
int a=5 // 0101
int c = a<<1 // 1010(왼쪽으로 한 칸이동)
```


## >> 
부호비트는 유지하고 오른쪽으로 이동한다.

```
int a = 5 //0101
int c = a>>1 //0010
```


## 부호없는 >>>
비트를 오른쪽으로 이동하고 빈칸을 0으로 메꾼다.

```
int a = -5;  // 11111111111111111111111111111011
int c = a >>> 1;  // 01111111111111111111111111111101 (2147483645)

```
