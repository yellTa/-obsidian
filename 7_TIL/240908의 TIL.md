# 자바 Generic
자바에서 맨날 보던 <>이걸 홀화살표라고 하더군요... 아무튼 
이친구는 컴파일 때 `<String>`여기 있는 String의 값으로 치환해주는 거 알고 계쎄요?..

그러니까 클래스가 

```java
class 배고파요<T>(T item){
	T 배고픔;
	public 배고파요(T item){
		this.배고픔= item;
	}
}
```

이 된답니다. 
그리고 T를 나중에 꺽새에 넣은 값으로 치환하죠 그리고 소거를 통해서 제거한답니다.이걸 타입소거라고 합니...다...
